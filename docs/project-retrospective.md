## 二、项目阶段概览

### 1. 初始可用阶段

- 将本地语音转文字工具打包成 macOS 菜单栏应用
- 支持基础热键触发、录音、自动粘贴
- 发布最早的 GitHub 版本 `v0.1.0`

### 2. 配置与交互增强阶段

- 引入模型参数设置 UI
- 引入热键设置、鼠标触发、提示音、文本清理等功能
- 对菜单和弹窗做中英文适配
- 持续优化对普通用户更友好的操作路径

### 3. 模型切换与识别质量提升阶段

- 从 SenseVoice 迁移到 Fun-ASR Nano 2512
- 引入运行时兼容处理和本地模型缓存优先策略
- 在识别准确率、英文热词保留、整体稳定性上明显提升

### 4. DMG 安装器产品化阶段

- 从源码安装转向 DMG 图形化安装
- 增加图形化安装窗口和图形化卸载器
- 集成 standalone Python、自动依赖安装、自动模型下载
- 补齐桌面快捷方式、启动器、卸载器、图标体系等产品化细节

### 5. 仓库重命名与正式发布阶段

- 仓库从旧命名迁移到 `FunASR-dictation-macos`
- 发布新仓库的 `v1.0.0`
- 继续围绕 macOS 14、权限、启动感知、自启动修复打磨稳定性
- 当前阶段已进入“可安装、可测试、可发布、可回归”的正式版本状态

## 三、功能改进清单

下面按功能维度整理我们在整个过程中新增或显著改进过的能力。

### 1. 菜单栏应用能力

- 将工具做成真正的 macOS 菜单栏应用
- 支持状态切换：
  - 就绪
  - 录音中
  - 转写中
  - 权限异常
  - 错误状态
- 支持启动时可见性优化与用户引导

### 2. 热键与触发方式

- 支持键盘热键触发
- 支持鼠标中键/扩展按键触发
- 将分散菜单合并为独立“快捷键设置”窗口
- 增加原生引导流程：
  - 开始识别按键
  - 手动输入
  - 当前设置展示

### 3. 模型参数配置

- 增加 `Model Config / 模型参数设置` 原生窗口
- 支持以下主要参数：
  - 识别语言
  - 采样率
  - 声道数
  - 粘贴延迟
  - 空闲卸载模型秒数
  - 数字与日期规范化
  - 合并长停顿片段
  - 过滤表情符号
  - 热词列表
- 后续又持续优化了：
  - 卡片式布局
  - 标题层级
  - 可滚动内容
  - 中英文适配

### 4. 文本处理体验

- 增加提示音控制
- 增加文本清理策略
- 优化中文间多余空格处理
- 优化标点前空格和括号内空格清理
- 保留热词原始大小写输入，不在保存阶段强制转小写

### 5. 模型与识别链路

- 从 SenseVoice 切换到 Fun-ASR Nano 2512
- 引入本地缓存优先模型加载
- 增加模型预热逻辑
- 增加空闲自动卸载模型以降低内存占用
- 支持再次使用时重新加载模型

### 6. 安装与卸载能力

- 支持 DMG 图形化安装
- 支持图形化卸载
- 安装时自动准备 standalone Python
- 安装时自动创建虚拟环境并安装依赖
- 安装时自动下载模型
- 支持可选桌面快捷方式
- 卸载时清理：
  - 应用本体
  - 卸载器
  - 虚拟环境
  - Python runtime
  - 模型缓存
  - LaunchAgent
  - 旧版目录

### 7. 启动与自启动能力

- 支持手动从 `Applications` 启动
- 支持开机自动启动
- 支持在权限异常时切换到可见模式，避免“后台无感运行”
- 增加启动浮层提示
- 增加“程序已在运行”时的二次点击提醒

### 8. 图标与品牌体系

- 应用名称统一为 `FunASR Dictation`
- 应用主图标改为新的渐变麦克风图标
- 卸载器图标独立设计并带垃圾桶标识
- 为旧版 macOS 自带圆角缺失问题补齐图标圆角，保证不同系统显示更一致

### 9. 多语言支持

- 菜单项支持中英文
- 主要弹窗支持中英文
- 安装器与卸载器支持中英文
- README 中英文双文档维护
- 增加运行时语言切换入口，方便测试英文菜单和弹窗

## 四、问题修复清单

以下按问题域整理我们一路修过的主要问题。

### 1. 模型安装与加载问题

- 修复 Fun-ASR Nano 模型“未注册”错误
- 修复 `trust_remote_code` 回退路径不稳的问题
- 修复模型首次下载后加载失败的问题
- 修复本地模型缓存存在时仍重复走网络解析的问题
- 修复模型加载失败后状态恢复不一致的问题

### 2. 识别结果问题

- 修复“录音结束后没有输出文本”的问题
- 修复转写完成但文本没有写入目标输入框的问题
- 修复粘贴延迟导致的偶发粘贴失败
- 修复部分多余空格问题
- 优化中文场景下中英混写的稳定性

### 3. 权限与 TCC 问题

- 修复辅助功能权限反复失效
- 修复输入监控权限链路不稳定
- 修复 launcher bundle identity 漂移导致的权限归属问题
- 增加权限错误后的用户提示和重启引导
- 修复权限异常时完全无反馈、用户误以为程序未启动的问题

### 4. 菜单栏与可见性问题

- 修复菜单栏图标不显示
- 修复菜单栏标题前误出现 `FA`
- 修复状态栏图标在不同系统上的可见性 fallback
- 增加启动浮层，减少“程序已启动但用户没看到图标”的情况
- 修复程序已在运行时再次点击图标没有任何反馈的问题

### 5. 启动器与启动链路问题

- 修复桌面快捷方式双击无反应
- 修复 launcher 执行链静默失败
- 修复启动路径写死、迁移后指向旧目录的问题
- 修复运行目录 metadata 缺失时无法自愈的问题
- 修复 autostart 配置损坏后系统仍显示“已开启”的问题

### 6. 自启动问题

- 修复 LaunchAgent 启用顺序问题
- 修复外接磁盘路径下的开机自启动问题
- 修复 autostart runner 缺失时不判定损坏的问题
- 将运行支持目录统一到 `FunASRDictation`，避免旧 `SenseVoiceDictation` 目录继续成为运行依赖

### 7. 安装器问题

- 修复安装器首次 bootstrap 失败
- 修复本地化 macOS 上找不到 Terminal 的问题
- 修复 DMG 安装后自动打开应用触发系统隐私扩展崩溃的问题
- 修复安装器无取消能力的问题
- 增加取消后的清理逻辑
- 修复 standalone Python 下载慢和 HTTP/2 失败问题
- 修复 Gatekeeper 对 `python3.11` 的拦截问题

### 8. 卸载器问题

- 修复卸载器点击 `Cancel` 后仍无法关闭窗口
- 修复桌面快捷方式删除失败会中断卸载的问题
- 增加卸载器本地化提醒
- 修复卸载器在 metadata 缺失时无法定位运行目录的问题

### 9. UI 细节与布局问题

- 多轮修复 `Model Config` 的：
  - 高度溢出
  - 内容裁切
  - 不可滚动
  - 卡片边距不一致
  - 标题与图标不居中
- 多轮修复 `Hotkey Settings` 的：
  - 按钮视觉层级
  - 标题与卡片间距
  - 当前设置字号
  - 左右对齐
- 安装器窗口多轮修正：
  - 文案截断
  - 中英文排版
  - 按钮区布局

## 六、阶段性结论

这次项目迭代的核心价值，不只是把一个语音听写工具“做出来”，而是把它从一个面向开发者的脚本工具，逐步打磨成一个普通 macOS 用户也可以安装、配置、使用和卸载的本地应用。

从结果看，我们完成了几件很关键的事情：

- 模型从 SenseVoice 平稳迁移到 Fun-ASR，提升了识别质量
- 把配置、热键、安装、卸载、权限提示等关键链路做成了真正的产品功能
- 在真实机器、真实系统版本、真实权限环境中持续修复问题
- 把大量“只有用户装到自己机器上才会暴露”的问题逐步收敛进测试与工程化体系

如果只用一句话概括这段过程，可以这样说：

> 我们把一个“本地听写脚本”，做成了一套真正可以交付给别人安装和使用的 macOS 产品。
